Проект собран на Delphi XE.
Включает в себя:
1. Project1.dpr, MainUnit.pas, MainUnit.dfm - основной модуль.
2. TaskLibrary.dpr - тестовая DLL (просто заполняет прогресс-бар за указанное время)
3. FileSearchDll3.dpr - DLL для поиска файлов по маске. Включает две задачи:
  - поиск файлов в указанной папке (маску файлов можно указывать через запятую)
  - поиск вхождений текстовых фрагментов в указанном файле (различные варианты текстовых фрагментов можно указывать через запятую. Поиск регистрочувствительный).
4. ShellCommandDLL.dpr - выполнение shell-команды.
